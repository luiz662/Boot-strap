# ⚡ TaskFlow - Landing Page

O **TaskFlow** é uma landing page moderna e responsiva desenvolvida para um aplicativo de gerenciamento de tarefas e produtividade. O projeto foi construído utilizando **Bootstrap 5**, focando em uma interface limpa, intuitiva e totalmente adaptável a diferentes tamanhos de tela.

---

## 🚀 Funcionalidades e Seções

* **Navegação Responsiva (`Navbar`):** Menu superior com links de ancoragem e botão de login integrado, adaptável para dispositivos móveis através de um menu colapsável (hambúrguer).
* **Seção Principal (`Hero Section`):** Apresentação do produto com chamada para ação (CTA) clara, botões interativos e espaço para mockup demonstrativo.
* **Seção de Recursos (`Features`):** Exibição dos principais diferenciais da plataforma (Kanban Boards, Time Tracking e Segurança) utilizando o sistema de *Grid* e *Cards* do Bootstrap.
* **Rodapé (`Footer`):** Informações de direitos autorais estruturadas de forma simples e elegante.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

* **HTML5:** Estruturação semântica da página.
* **Bootstrap v5.3.x:** Framework CSS utilizado para agilizar o estilizado e garantir total responsividade.
* **Bootstrap Icons v1.13.1:** Biblioteca de ícones vetoriais.
* **CSS3 (Customizado):** Para ajustes finos adicionais (`css/style.css`).
